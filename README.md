# Audio-Processing-for-Meeting
Nội dung Repository
Pipeline.ipynb - Quy trình chính
Xử lý audio toàn diện từ đầu vào đến đầu ra
Áp dụng các bộ lọc và xử lý tín hiệu
Tạo bản audio đã xử lý

Finetune.ipynb - Tinh chỉnh Mô hình
Huấn luyện/tinh chỉnh mô hình machine learning
Sử dụng dữ liệu audio từ cuộc họp
Tối ưu hóa hiệu suất mô hình

Compare.ipynb - So Sánh Kết Quả
Đánh giá các phương pháp xử lý khác nhau
Phân tích chất lượng audio trước/sau
Đo lường hiệu suất các mô hình

Yêu Cầu
Python 3.7+
Jupyter Notebook
Các thư viện: librosa, numpy, scipy, matplotlib, sklearn (xem chi tiết trong notebook)

Cài Đặt
# Clone repository
git clone https://github.com/LamBaThanh/Audio-Processing-for-Meeting.git
cd Audio-Processing-for-Meeting

# Cài đặt các dependencies
pip install -r requirements.txt  # Nếu có tệp này

# Khởi chạy Jupyter
jupyter notebook
